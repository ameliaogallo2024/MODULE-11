# Module 11 Assignment: Data Visualization with Matplotlib


**SunCoast Retail Visual Analysis**

## Description
This project focuses on the visual exploration of retail performance data for "SunCoast Retail." Using Python and Matplotlib, the analysis transforms complex quarterly data from multiple Florida locations (Tampa, Miami, Orlando, and Jacksonville) into intuitive visual narratives. The project examines sales growth, category performance, and the relationship between marketing investment and revenue.

## Files Included
* `suncoast_retail_analysis.py`: The Python script containing the data generation logic and five distinct visualization functions.
* `README.md`: Documentation for the repository.

## What I Practiced
In this assignment, I advanced my technical skills in data storytelling and library management:
* **Advanced Plotting:** I practiced creating various chart types, including line plots for trends, grouped bar charts for comparison, and stacked bar charts for composition analysis.
* **Pandas Data Shaping:** I used `pivot_table` and `unstack` to restructure data specifically for visualization requirements.
* **Time-Series Handling:** I practiced working with `pd.date_range` and handled the transition from deprecated frequency aliases (updating 'Q' to 'QE') to ensure code longevity.
* **Aesthetic Customization:** I refined plot layouts using `tight_layout`, legend positioning (`bbox_to_anchor`), and grid styling to improve professional readability.
* **Business Insights:** I analyzed specific retail factors such as seasonal weighting (Q4 surges) and location-specific category preferences.
